# Ember & Oak — Demo Site

> Portfolio demonstration project. All content is fictional, no real client data was used.

## Overview
A fully responsive website for a fictional premium building & design materials brand, 
built to showcase frontend development skills in a lifestyle/commercial context.

## Features
- Responsive design (mobile & desktop)
- Mobile hamburger menu with animated icon toggle
- Sticky navbar with scroll blur effect
- Hero section with Ken Burns zoom animation
- USPs bar (Direct Manufacturer / Supplier highlights)
- Products grid with hover zoom effect
- Scroll reveal animations (Intersection Observer)
- Active link highlighting on scroll
- Payment & Delivery options section
- Structured Data (Schema.org) for SEO

## Tech Stack
- HTML5
- CSS3
- JavaScript (Vanilla)
- Font Awesome 6 (icons)
- Google Fonts (Outfit)
